# 🚢 Titanic Intelligence Hub - Dashboard Interactif
<img width="1918" height="831" alt="image" src="https://github.com/user-attachments/assets/34453688-01bc-4dc0-ad63-dd5ea2b37eda" />


Ce projet est un dashboard d'analyse de données interactif focalisé sur la tragédie du Titanic en 1912. 
Il permet d'explorer les statistiques de survie, la démographie des passagers et les corrélations socio-économiques à travers des visualisations modernes et dynamiques.

## 🌐 Lien du Projet
> **Lien direct :** [Consulter le Dashboard en direct](https://votre-pseudo.github.io/votre-depot/)

##  Aperçu
Le dashboard est conçu comme une application monopage (SPA) utilisant des onglets pour séparer les différentes dimensions de l'analyse :
- **Analyses Globales** : Taux de survie, répartition par classe, ports d'embarquement et genre.
- **Démographie & Social** : Profil social des rescapés (Radar Chart) et pyramide des âges.

##  Fonctionnalités
- **KPIs en temps réel** : Affichage permanent des indicateurs clés (Taux de survie, moyenne d'âge, prix du billet).
- **Navigation par onglets** : Interface fluide pour passer d'une vue à l'autre sans rechargement.
- **Visualisations avancées** : Utilisation de Chart.js pour des graphiques variés (Donut, Barres empilées, Radar, Courbes).
- **Design "Dark Mode"** : Interface moderne optimisée pour la lisibilité.

##  Technologies utilisées
* **HTML5** : Structure sémantique du dashboard.
* **CSS3** : Mise en page via **CSS Grid** pour une symétrie parfaite des graphiques et design responsive.
* **JavaScript (ES6+)** : Logique de changement d'onglets et gestion dynamique du DOM.
* **Chart.js** : Bibliothèque de rendu de graphiques haute performance.

##  Sources des données
Les données utilisées pour ce dashboard proviennent de sources historiques et scientifiques reconnues :
* **Encyclopedia Titanica**
* **British Wreck Commissioner's Inquiry (1912)**
* **Kaggle Dataset** (Titanic: Machine Learning from Disaster)
